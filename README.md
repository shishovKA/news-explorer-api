# Диплом Бэкенд.

### Описание работы:
Подготовка и деплой бэкенда

### Инструкция по запуску:
Запуск проекта:
*  npm run start - запускаем сервер на localhost:3000
*  npm run dev - запускаем сервер на localhost:3000 с хот релоудом
